## 🎓 5-Minute Tutorial

### Step 1: Learn the Basics (2 minutes)
1. Click "📖 Learn: Stock Basics"
2. Read the "Momentum Strategy" tab
3. Understand: **Momentum Score = ROC × Trend Slope**

### Step 2: Try the Calculator (2 minutes)
1. Click "🎯 Momentum Strategy"
2. Enter a stock ticker (e.g., "AAPL")
3. Click "Calculate Momentum"
4. See the results and interpretation

### Step 3: Analyze Multiple Stocks (1 minute)
1. Click "📊 Live Market Analysis"
2. Select "Tech Giants"
3. Click "Analyze Stocks"
4. See which stocks have the strongest momentum

**Congratulations!** You've just learned how professional momentum strategies work! 🎉

## 💡 Example Walkthrough

### Analyzing Apple (AAPL)

1. **Go to Momentum Strategy page**

2. **Enter ticker:** `AAPL`

3. **Adjust parameters:**
   - Lookback Period: 200 days
   - ROC Period: 200 days
   - Data Period: 1 year

4. **Click "Calculate Momentum"**

5. **Interpret results:**
   - **Positive score** = Stock is trending up
   - **High ROC** = Strong recent gains
   - **Positive slope** = Sustained uptrend
   - **High annualized return** = Strong momentum

6. **Look at the chart:**
   - Price above MA50 and MA200 = Bullish
   - MA50 above MA200 = Strong uptrend
   - Rising prices = Momentum confirmed

### Comparing Multiple Stocks

1. **Go to Live Market Analysis**

2. **Select "Tech Giants"**
   - Includes: AAPL, MSFT, GOOGL, AMZN, META, NVDA, TSLA

3. **Click "Analyze Stocks"**

4. **Review results:**
   - Top 3 stocks shown in cards
   - Full table with all metrics
   - Bar chart for visual comparison

5. **Understand rankings:**
   - Highest momentum score = Strongest trend
   - Positive ROC = Recent gains
   - Compare across stocks

## 🔍 Understanding the Results

### Momentum Score
- **> 0.01**: Strong positive momentum
- **0 to 0.01**: Weak positive momentum
- **< 0**: Negative momentum (avoid)

### ROC (Rate of Change)
- **> 20%**: Very strong gains
- **10-20%**: Strong gains
- **0-10%**: Moderate gains
- **< 0%**: Losses (avoid)

### Trend Slope
- **> 0.001**: Strong uptrend
- **0 to 0.001**: Weak uptrend
- **< 0**: Downtrend (avoid)

### Annualized Return
- **> 50%**: Exceptional momentum
- **20-50%**: Strong momentum
- **0-20%**: Moderate momentum
- **< 0%**: Negative momentum

## 🐛 Troubleshooting

### App Won't Start
```bash
# Check Python version (need 3.9+)
python --version

# Reinstall dependencies
pip install -r requirements.txt

# Try again
streamlit run app.py
```

### Data Not Loading
- Check internet connection
- Try a different stock ticker
- Wait a moment and retry

### Slow Performance
- Close other browser tabs
- Restart the app
- Use shorter time periods

### Error Messages
- Read the error carefully
- Check ticker symbol is correct
- Ensure you have internet access
- Try a different stock

# 📚 Stock Learning Hub

> **Interactive platform to learn stock investment through proven strategies**

## ⚡ Quick Start

### Windows:
```bash
run.bat
```

### Mac/Linux:
```bash
chmod +x run.sh
./run.sh
```

**Opens automatically at:** `http://localhost:8501`

---

## 📖 What's Inside

### 7 Learning Modules:
1. **🏠 Home** - Overview
2. **📖 Learn Basics** - Stock fundamentals
3. **🎯 Momentum Strategy** - Calculate momentum scores
4. **📊 Live Analysis** - Compare multiple stocks
5. **🔬 Backtest** - Test strategies
6. **🚀 Advanced** - 4 professional strategies
7. **⚔️ Comparison** - Compare all strategies

### 4 Professional Strategies:
1. **Top-N Momentum TAA** - Select best performers
2. **60/40 Portfolio** - Balanced approach
3. **ROC × Trend Slope** - Advanced momentum
4. **Indicator-Based** - Technical analysis (RSI, MACD, MA)

---

## 🎓 Learning Path

1. **Start:** Read "Learn: Stock Basics" (15 min)
2. **Practice:** Calculate momentum for AAPL, MSFT (5 min)
3. **Analyze:** Compare Tech Giants (5 min)
4. **Compare:** Run strategy comparison (5 min)

**Total: 30 minutes to understand everything!**

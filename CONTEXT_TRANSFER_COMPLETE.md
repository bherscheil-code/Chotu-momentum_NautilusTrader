## 🚀 4 Professional Trading Strategies

### 1. Top-N Momentum TAA
- **From**: Chotu-qstrader
- **Logic**: Selects top N assets by holding-period return
- **Best for**: Bull markets, trending stocks
- **Parameters**: Lookback period, Number of assets

### 2. 60/40 Portfolio
- **From**: Chotu-qstrader
- **Logic**: Classic balanced portfolio (60% stocks, 40% bonds)
- **Best for**: Balanced risk/return
- **Parameters**: Stock/Bond allocation

### 3. ROC × Trend Slope
- **From**: Root momentum files (your proven backtests)
- **Logic**: Advanced momentum = ROC × Trend Slope
- **Best for**: Strong trends, systematic trading
- **Parameters**: Lookback, ROC period, Top N

### 4. Indicator-Based
- **From**: Chotu-stock-analysis-engine
- **Logic**: Multi-indicator confirmation (RSI, MACD, MA)
- **Best for**: Technical analysis, multiple signals
- **Parameters**: Min buy/sell indicators

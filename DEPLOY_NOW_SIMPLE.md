## 📋 3 Simple Steps

### Step 1: GitHub (2 minutes)

1. Go to: **https://github.com/new**
2. Repository name: `stock-learning-hub`
3. Make it **Public**
4. Click **"Create repository"**

### Step 2: Push Code (2 minutes)

Open terminal in your project folder and run:

```bash
git init
git add .
git commit -m "Stock Learning Hub"
git remote add origin https://github.com/YOUR_USERNAME/stock-learning-hub.git
git branch -M main
git push -u origin main
```

**Replace `YOUR_USERNAME` with your GitHub username!**

### Step 3: Deploy (3 minutes)

1. Go to: **https://share.streamlit.io**
2. Click **"Sign in with GitHub"**
3. Click **"New app"**
4. Fill in:
   - Repository: `YOUR_USERNAME/stock-learning-hub`
   - Branch: `main`
   - Main file: `Stock-Learning-Hub/app.py`
5. Click **"Deploy!"**

**Wait 2-3 minutes... Done!**
